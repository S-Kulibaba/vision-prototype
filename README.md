# Vision Prototype

A local tool for detecting and analyzing animation in video files, built as a proof of concept.

Given a video file, the tool automatically:

- Detects animated vs. live-action segments with timestamps
- Transcribes speech and attaches it to each segment
- Generates a concise research report using a local language model

---

## How it works

```
Video file
    │
    ├── Whisper          → speech transcription with word-level timestamps
    │
    ├── CLIP + OpenCV    → per-frame classification (animation vs. live action)
    │       ├── CLIP semantic score
    │       └── Visual features: noise level, edge density, color count
    │
    ├── Smoothing        → confidence-weighted sliding window to remove flicker
    │
    ├── Segmentation     → timed segments with transcripts attached
    │
    └── Mistral (local)  → structured research report
```

---

## Stack

| Component                                    | Purpose                       |
| -------------------------------------------- | ----------------------------- |
| [CLIP](https://github.com/openai/CLIP)       | Semantic frame classification |
| [OpenCV](https://opencv.org/)                | Visual feature extraction     |
| [Whisper](https://github.com/openai/whisper) | Speech-to-text transcription  |
| [Ollama](https://ollama.com/) + Mistral 7B   | Local report generation       |
| Flask                                        | Web interface                 |

All models run **locally** with no API keys. No data leaves your machine.

## Requirements

- Python 3.10+
- NVIDIA GPU recommended (CUDA 12.1). CPU works, but is slower.
- [Ollama](https://ollama.com/) installed separately
- [ffmpeg](https://ffmpeg.org/) installed and available in PATH

---

## Installation

**1. Clone the repository**

```bash
git clone https://github.com/YOUR_USERNAME/vision-prototype.git
cd vision-prototype
```

**2. Create a virtual environment**

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

**3. Install Python dependencies**

```bash
# GPU (CUDA 12.1) — recommended
pip install torch==2.5.1+cu121 torchvision==0.20.1+cu121 --index-url https://download.pytorch.org/whl/cu121

# CPU only
pip install torch torchvision

# Then install the rest
pip install -r requirements.txt
```

> If you don't have a CUDA-capable GPU, follow CPU-only installation.

**4. Install Mistral via Ollama**

Make sure Ollama is running, then:

```bash
ollama pull mistral
```

**5. Install ffmpeg**

- Windows: download from [gyan.dev](https://www.gyan.dev/ffmpeg/builds/), extract, and add the `bin` folder to your PATH
- macOS: `brew install ffmpeg`
- Linux: `sudo apt install ffmpeg`

---

## Usage

```bash
python analyze.py
```

Open [http://localhost:5000](http://localhost:5000) in your browser.

Drag and drop a video file (MP4, AVI, MOV) onto the upload zone. The tool will process it and display:

- A timeline of animation and live-action segments
- Speech transcript attached to each segment
- A research report generated by Mistral
